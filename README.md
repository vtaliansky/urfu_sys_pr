# Пример SRS для лабораторной работы

## Вариант 12. Информационная система гостиничного комплекса

Гостиничный комплекс состоит из нескольких зданий-гостиниц. Система предназначена для учета корпусов, номеров, бронирований, договоров с организациями, проживания постояльцев, дополнительных услуг, задолженностей, отзывов, жалоб, расходов и доходов гостиничного комплекса.
