### Диаграмма деятельности для варианта использования "2. Ведение договоров с организациями"

```plantuml
@startuml

title Диаграмма деятельности для варианта использования "2. Ведение договоров с организациями"

start
:Руководитель открывает раздел "Организации и договоры"; <<Пользователь>>
:Система показывает список организаций и договоров; <<Система>>
if (Организация уже есть в системе?) then (да)
  :Руководитель выбирает организацию; <<Пользователь>>
else (нет)
  :Руководитель вводит данные организации; <<Пользователь>>
  :Система сохраняет организацию; <<Система>>
endif
:Руководитель вводит условия договора, срок действия и скидку; <<Пользователь>>
:Система проверяет даты и размер скидки; <<Система>>
if (Договор корректен?) then (да)
  :Система сохраняет договор; <<Система>>
  :Система делает договор доступным для группового бронирования; <<Система>>
else (нет)
  :Система показывает ошибки; <<Система>>
endif
stop
@enduml
```
