### Диаграмма деятельности для варианта использования "4. Заселение и выселение постояльца"

```plantuml
@startuml

title Диаграмма деятельности для варианта использования "4. Заселение и выселение постояльца"

start
:Администратор открывает раздел "Заселение/выезд"; <<Пользователь>>
if (Постоялец заселяется?) then (да)
  :Администратор ищет бронь или свободный номер; <<Пользователь>>
  :Система показывает данные брони или свободные номера; <<Система>>
  if (Клиент новый?) then (да)
    :Администратор вводит данные клиента; <<Пользователь>>
    :Система создает карточку клиента; <<Система>>
  else (нет)
    :Система использует существующую карточку клиента; <<Система>>
  endif
  :Администратор подтверждает заселение; <<Пользователь>>
  :Система создает проживание и меняет статус номера на "занят"; <<Система>>
else (нет)
  :Администратор выбирает проживание для выезда; <<Пользователь>>
  :Система рассчитывает проживание, услуги и задолженность; <<Система>>
  :Администратор фиксирует оплату или долг; <<Пользователь>>
  :Система закрывает проживание и освобождает номер; <<Система>>
endif
stop
@enduml
```
