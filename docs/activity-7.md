### Диаграмма деятельности для варианта использования "7. Учет расходов и доходов гостиничного комплекса"

```mermaid
flowchart TD
    A([Начало]) --> B{Администратор вводит расход?}
    B -- Да --> C[Администратор выбирает пункт меню "Расходы"]
    C --> D[Система отображает форму расхода]
    D --> E[Администратор вводит статью, дату, сумму и комментарий]
    E --> F[Система сохраняет расход]
    F --> Z([Конец])
    B -- Нет, руководитель смотрит отчет --> G[Руководитель выбирает пункт меню "Отчеты"]
    G --> H[Система отображает список отчетов]
    H --> I[Руководитель выбирает период и тип отчета]
    I --> J[Система собирает данные из броней, проживаний, услуг, оплат, расходов и жалоб]
    J --> K[Система формирует отчет: свободные номера, популярность, доходы, расходы, долги, отзывы и жалобы]
    K --> L[Руководитель просматривает отчет]
    L --> Z
```
