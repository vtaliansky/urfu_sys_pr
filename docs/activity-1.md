### Диаграмма деятельности для варианта использования "1. Ведение номерного фонда и тарифов"

```mermaid
flowchart TD
    A([Начало]) --> B[Администратор выбирает пункт меню "Номерной фонд"]
    B --> C[Система отображает список корпусов и номеров]
    C --> D{Нужно добавить или изменить корпус?}
    D -- Да --> E[Администратор вводит данные корпуса: класс отеля, этажи, комнаты, службы, питание, развлечения]
    E --> F[Система проверяет обязательные поля]
    F --> G[Система сохраняет карточку корпуса]
    D -- Нет --> H{Нужно добавить или изменить номер?}
    G --> H
    H -- Да --> I[Администратор вводит данные номера: корпус, этаж, вместимость, тип, стоимость, статус]
    I --> J[Система проверяет, что номер не дублируется в корпусе]
    J --> K[Система сохраняет карточку номера]
    H -- Нет --> L[Система обновляет список номерного фонда]
    K --> L
    L --> M([Конец])
```
