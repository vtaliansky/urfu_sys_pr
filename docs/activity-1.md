### Диаграмма деятельности для варианта использования "1. Ведение данных гостиничного комплекса"

```plantuml
@startuml

title Диаграмма деятельности для варианта использования "1. Ведение данных гостиничного комплекса"

start
:Администратор или руководитель открывает раздел "Номера и услуги"; <<Пользователь>>
:Система показывает список корпусов, номеров, услуг и тарифов; <<Система>>
if (Нужно добавить новый объект?) then (да)
  :Пользователь нажимает "Добавить"; <<Пользователь>>
  :Система открывает форму создания; <<Система>>
else (нет)
  :Пользователь выбирает существующий объект; <<Пользователь>>
  :Система открывает форму редактирования; <<Система>>
endif
:Пользователь заполняет данные корпуса, номера, услуги или тарифа; <<Пользователь>>
:Система проверяет обязательные поля; <<Система>>
if (Данные корректны?) then (да)
  :Система сохраняет изменения; <<Система>>
  :Система обновляет справочники для бронирования; <<Система>>
else (нет)
  :Система показывает ошибку заполнения; <<Система>>
endif
stop
@enduml
```
