### Диаграмма деятельности для варианта использования "3. Бронирование номеров"

```plantuml
@startuml

title Диаграмма деятельности для варианта использования "3. Бронирование номеров"

start
:Администратор, постоялец или представитель организации открывает форму бронирования; <<Пользователь>>
:Пользователь вводит даты, класс отеля, этаж, количество комнат и людей; <<Пользователь>>
if (Бронирование от организации?) then (да)
  :Система проверяет действующий договор и скидку; <<Система>>
else (нет)
endif
:Система ищет свободные номера на выбранный период; <<Система>>
if (Это групповое бронирование?) then (да)
  :Система сначала ищет номера на одном этаже; <<Система>>
else (нет)
endif
if (Подходящие номера найдены?) then (да)
  :Система показывает варианты размещения и стоимость; <<Система>>
  :Пользователь подтверждает бронь; <<Пользователь>>
  :Система сохраняет бронь и меняет статус номеров на "забронирован"; <<Система>>
else (нет)
  :Система сообщает, что подходящих номеров нет; <<Система>>
endif
stop
@enduml
```
