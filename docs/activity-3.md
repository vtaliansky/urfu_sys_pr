### Диаграмма деятельности для варианта использования "3. Бронирование номеров"

```mermaid
flowchart TD
    A([Начало]) --> B[Клиент, представитель организации или администратор передает параметры брони]
    B --> C[Система отображает форму бронирования]
    C --> D[Пользователь указывает даты, класс отеля, этаж, количество комнат и людей]
    D --> E[Система ищет свободные номера по заданным параметрам]
    E --> F{Свободных номеров достаточно?}
    F -- Нет --> G[Система сообщает, что подходящих номеров недостаточно]
    G --> Z([Конец])
    F -- Да --> H{Бронь групповая?}
    H -- Да --> I[Система пытается подобрать номера на одном этаже]
    I --> J{Один этаж доступен?}
    J -- Да --> K[Система предлагает номера на одном этаже]
    J -- Нет --> L[Система предупреждает, что часть группы придется расселить по разным этажам]
    H -- Нет --> M[Система рассчитывает стоимость]
    K --> M
    L --> M
    M --> N[Система применяет скидку по договору, если она есть]
    N --> O[Пользователь подтверждает бронь]
    O --> P[Система сохраняет бронь со статусом "Активна"]
    P --> Z
```
