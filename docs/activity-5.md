### Диаграмма деятельности для варианта использования "5. Учет дополнительных услуг и задолженности"

```plantuml
@startuml

title Диаграмма деятельности для варианта использования "5. Учет дополнительных услуг и задолженности"

start
:Администратор открывает карточку проживающего постояльца; <<Пользователь>>
:Система показывает активное проживание и текущую задолженность; <<Система>>
if (Нужно добавить услугу?) then (да)
  :Администратор выбирает услугу и количество; <<Пользователь>>
  :Система рассчитывает сумму начисления; <<Система>>
  :Система добавляет начисление в счет постояльца; <<Система>>
else (нет)
endif
if (Постоялец оплачивает долг?) then (да)
  :Администратор вводит сумму оплаты; <<Пользователь>>
  :Система фиксирует оплату и уменьшает задолженность; <<Система>>
else (нет)
  :Система оставляет задолженность открытой; <<Система>>
endif
stop
@enduml
```
